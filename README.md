# vue-components-library-boilerplate

## stack

- [x] Vue@^3
- [x] Vite@^2 library node
- [ ] TypeScript compile `typings` files
- [x] postcss@^9
- [x] tailwindcss@^2
- [x] storybook@^6

## startup

``` bash
$ npm i

# dev with storybook
$ npm run dev

# build with vite lib mode
$ npm run build

# build storybook static site
$ npm run build:book

# preview bundle content in '/preview' with vite
$ npm preview
```
